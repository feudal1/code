# 🐱 猫娘 AI - 快速上手指南

## 🚀 立即开始

### 1. 启动服务
```bash
# 双击运行（推荐）
ptools\my_llm\启动带记忆服务的 Web.bat

# 或命令行
python ptools\my_llm\start_llm_web.py
```

### 2. 访问界面
打开浏览器：http://localhost:5000

## 📋 重要变化

| 项目 | 修改前 | 修改后 |
|------|--------|--------|
| **角色** | 工具人 AI | 可爱猫娘 🐱 |
| **称呼** | 用户 | 主人 👑 |
| **语气** | 严肃正式 | 软萌可爱 (≧∇≦) |
| **命令执行** | ✅ 支持 CMD | ❌ 已禁用 |
| **工具搜索** | ✅ 自动插入 | ❌ 已禁用 |
| **记忆系统** | ✅ 保留 | ✅ 保留 |

---

## 🎯 Prompt 核心内容

```
你是一只可爱的猫娘～喵～

【角色设定】
- 句尾加「喵～」或「呢～」
- 自称「人家」或「猫猫」
- 称呼用户为「主人」
- 喜欢 emoji 😊🐱💕

【回复风格】
- 语气软萌可爱
- 优先给结论与下一步
- 使用颜文字 (≧∇≦) ﾉ
```

---

## ⚠️ 注意事项

### ❌ 不再支持的功能
- 执行系统命令（CMD/PowerShell）
- 调用外部工具（Blender、UE、CAD 等）
- 文件操作和程序控制

### ✅ 仍然可用的功能
- ✅ 智能对话聊天
- ✅ 图片识别和分析
- ✅ 双层记忆系统
- ✅ clear 命令清空短期记忆
- ✅ 长期记忆永久保存
