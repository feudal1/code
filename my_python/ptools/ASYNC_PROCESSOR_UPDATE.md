## 📋 使用方法

### 启动处理器
```bash
cd e:\code\my_python\ptools
python my_llm\async_processor.py
```

### 处理消息
- 输入 `1`：处理所有待处理消息（最多 50 条）
- 输入 `2`：处理前 10 条消息
- 输入 `3`：处理前 20 条消息
- 输入 `s`：查看统计信息
- 输入 `q`：退出程序

### 用户端操作
- 在网页上发送消息或图片
- 发送 `clear` 清空短期记忆
- 点击"刷新状态"查看 AI 回复

## 🐛 常见问题

### Q1: 连接超时怎么办？
**A:** Cookie 可能已过期，需要重新获取：
1. 浏览器访问：`https://feudal.fwh.is/get_pending_messages.php?limit=1`
2. F12 -> Network -> 找到请求 -> 复制 Cookie
3. 更新 `async_processor.py` 中的 Cookie 值

### Q2: Ollama 返回空响应？
**A:** 检查：
1. Ollama 服务是否运行：`ollama list`
2. 模型是否已下载：`ollama pull qwen3-vl:8b`
3. 查看错误日志了解具体原因

### Q3: 回复出现重复循环？
**A:** 上下文过长导致，现在已修复：
- 只保留最近 10 条对话
- 图片数据不保存到记忆
- 自动清理过大的记忆文件

### Q4: 如何清空记忆？
**A:** 
- **用户端**：在网页发送 `clear` 消息
- **管理员**：运行 `python my_llm\cleanup_memory.py`
