# Clear 命令使用说明

## 🎯 正确的使用方式

### ❌ 错误做法
在处理器命令行中输入 `clear`
```
你的选择：clear
⚠️ 无效输入，请重新选择
```

### ✅ 正确做法

#### 步骤 1：用户在网页发送 clear
1. 打开浏览器访问：`https://feudal.fwh.is/`
2. 在输入框中输入：`clear`
3. 点击"发送"按钮

#### 步骤 2：运行处理器处理消息
```bash
cd e:\code\my_python\ptools
python my_llm\async_processor.py
```

在处理器中选择：
```
你的选择：1
```

#### 步骤 3：查看结果
处理器会显示：
```
📨 处理消息 ID: XX, 用户：user_xxx
📝 内容：clear
   调试：原始文本='clear', 处理后='clear'
🧹 检测到 clear 命令，清空短期记忆
✅ 短期记忆已清空喵～人家重新开始和主人聊天啦！(≧∇≦) ﾉ
✅ clear 命令处理完成
```

#### 步骤 4：用户刷新网页
在网页上点击"🔄 刷新状态"按钮，就能看到 AI 的回复了。

## 📋 其他特殊命令

目前支持的特殊命令：
- `clear` - 清空短期记忆

未来可以添加更多命令，例如：
- `help` - 显示帮助信息
- `status` - 显示当前状态
- `memory` - 显示记忆内容

## 💡 提示

- **处理器命令行**只接受：`1`, `2`, `3`, `s`, `q`
- **网页消息**可以是任何内容，包括特殊命令如 `clear`
- 特殊命令会在处理器处理时被识别和执行
