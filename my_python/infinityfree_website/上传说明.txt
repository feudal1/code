# 📤 InfinityFree 上传指南

## ⚠️ 重要提示

InfinityFree **只允许上传到 htdocs 文件夹**，这是安全要求。

## ✅ 正确的上传方法

### 方法 1: 直接上传文件（推荐）

1. **打开 FTP 客户端**（如 FileZilla）
2. **连接到服务器**
   - 主机：ftpupload.net（或控制面板提供的 FTP 主机）
   - 用户名：你的 InfinityFree 用户名
   - 密码：你的密码

3. **进入 htdocs 目录**
   ```
   /htdocs/
   ```

4. **选择以下文件并上传**（不要上传文件夹本身）
   ```
   config.php
   database.sql
   receive.php
   check_status.php
   get_pending_messages.php
   submit_response.php
   index.html
   check_config.php
   README.md
   QUICKSTART.md
   PROJECT_INFO.md
   ```

5. **确认上传后**，访问 `https://your-domain.infinityfreeapp.com/check_config.php` 验证

### 方法 2: 使用在线文件管理器

1. 登录 InfinityFree 控制面板
2. 点击 "File Manager"
3. 进入 `htdocs` 目录
4. 逐个上传文件

## 📁 项目结构说明

```
infinityfree_website/          ← 这个文件夹不要上传！
├── config.php                 ✓ 上传此文件到 htdocs
├── database.sql               ✓ 上传此文件到 htdocs
├── receive.php                ✓ 上传此文件到 htdocs
├── check_status.php           ✓ 上传此文件到 htdocs
├── get_pending_messages.php   ✓ 上传此文件到 htdocs
├── submit_response.php        ✓ 上传此文件到 htdocs
├── index.html                 ✓ 上传此文件到 htdocs
├── check_config.php           ✓ 上传此文件到 htdocs
└── README.md                  (可选)
```

## ❌ 错误的做法

```
❌ 不要这样做：
/htdocs/infinityfree_website/config.php
/htdocs/infinityfree_website/receive.php
...

✅ 应该这样做：
/htdocs/config.php
/htdocs/receive.php
...
```

## 🔍 验证安装

上传完成后，访问以下 URL 验证：

1. **配置检查**: `https://your-domain.infinityfreeapp.com/check_config.php`
2. **前端页面**: `https://your-domain.infinityfreeapp.com/`
3. **API 测试**: `https://your-domain.infinityfreeapp.com/get_pending_messages.php?limit=1`

## 💡 提示

- 如果看到 "Upload failed" 错误，请检查是否直接上传到了 `htdocs` 目录
- 确保没有将整个 `infinityfree_website` 文件夹上传
- 可以分批上传文件，避免网络中断
