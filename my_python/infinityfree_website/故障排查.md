# 🔧 故障排查指南 - Connection timed out

## ❌ 错误信息

```
❌ 发送失败：Connection timed out
```

## 💡 问题原因

这个错误表示 **Python 程序无法连接到 InfinityFree 网站**，原因可能是：

1. ✅ **网站还未部署** - PHP 文件未上传到 InfinityFree
2. ✅ **BASE_URL 配置错误** - 使用了默认的假域名
3. ✅ **网络问题** - 无法访问 InfinityFree

---

## ✅ 解决方案

### 方案 1: 如果你还没有部署网站

**重要提示：电脑端处理器需要一个已经部署好的 InfinityFree 网站才能工作！**

#### 步骤 1: 注册 InfinityFree
1. 访问 https://infinityfree.net
2. 注册免费账号
3. 创建网站，获得域名（如：`yourname.infinityfreeapp.com`）

#### 步骤 2: 上传 PHP 文件
使用 FTP（FileZilla）或在线文件管理器，将以下文件上传到 `/htdocs` 目录：
- config.php
- database.sql
- receive.php
- check_status.php
- get_pending_messages.php
- submit_response.php
- index.html
- check_config.php

#### 步骤 3: 配置数据库
编辑 `config.php`，填入从 InfinityFree 控制面板获取的数据库信息。

#### 步骤 4: 创建数据表
在 phpMyAdmin 中执行 `database.sql`。

#### 步骤 5: 验证部署
访问：`https://yourname.infinityfreeapp.com/check_config.php`

---

### 方案 2: 如果网站已部署但配置错误

#### 修改 BASE_URL

编辑文件：**`ptools/my_llm/async_processor.py`**

找到第 21 行：
```python
BASE_URL = os.getenv("INFINITYFREE_URL", "https://your-domain.infinityfreeapp.com")
```

修改为你的实际域名：
```python
BASE_URL = "https://yourname.infinityfreeapp.com"  # 替换为你的真实域名
```

## 💬 常见疑问

### Q: 我可以不部署网站直接使用吗？
**A:** ❌ 不可以。InfinityFree 网站作为云端数据库，用于存储待处理和已处理的消息。没有网站，消息无处存储。

### Q: 我可以在本地测试吗？
**A:** 可以，但需要：
1. 安装 PHP 和 MySQL（如使用 XAMPP）
2. 配置本地服务器
3. 修改 BASE_URL 为 `http://localhost`

### Q: 为什么需要联网？
**A:** 
- 网站托管在 InfinityFree（云端）
- 电脑端需要从云端获取消息
- AI 处理在本地完成（不需要联网调用 API）

### Q: 能不能完全离线使用？
**A:** 如果要完全离线：
1. 需要在本地搭建 PHP+MySQL 环境
2. 或者改用其他本地数据库方案
